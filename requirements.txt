📌 Requirements for Cup and Ball Shuffle Game
This project is a simple frontend-based game built using HTML, CSS, and JavaScript, so it doesn't require complex installations or backend setups.

✅ 1. System Requirements
💻 Operating System: Windows / macOS / Linux
🌐 Browser: Google Chrome, Mozilla Firefox, Edge, or Safari
🛠 Code Editor: VS Code, Sublime Text, or any text editor

✅ 2. Required Technologies & Dependencies
No external libraries are needed—just vanilla JavaScript, HTML, and CSS!

However, if you want to improve the game, you can include:
🔹 GSAP (GreenSock Animation Platform) – For smoother animations
🔹 Bootstrap / Tailwind CSS – For styling enhancements
🔹 Local Storage – To track high scores

To install GSAP (optional):

npm install gsap

✅ 4. Running the Game Locally
